# XMD

> docx to markdown and back again




## Install

I have no idea how I am going to distribute this yet.


## Usage

```sh
# docx to markdown
$ xmd tomd a.docx b.md

# markdown to docx
$ xmd tox b.md a.docx

# add git hook to local repository
$ xmd git init
```
